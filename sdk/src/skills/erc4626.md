# ERC-4626

Standard interface for tokenized vaults (EIP-4626). Provides deposit, withdraw, mint, redeem and preview functions for any conforming vault.

## Category
yield | Chains: (standard interface, any chain)

## Key Operations
- **deposit**: Deposit underlying assets, receive vault shares
- **withdraw**: Withdraw by specifying asset amount to receive
- **redeem**: Withdraw by specifying share amount to burn
- **mint**: Mint exact number of shares by depositing required assets
- **previewDeposit**: Preview how many shares a deposit would yield
- **previewWithdraw**: Preview how many shares would be burned for a withdrawal
- **convertToShares**: Convert asset amount to equivalent shares
- **convertToAssets**: Convert share amount to equivalent assets

## SauceScript Examples
```typescript
// Deposit assets into vault
import { VaultABI as IVault } from "./abis";
function main(vaultAddress: Address, assets: Uint256, receiver: Address): Uint256 {
  const vault = IVault.at(vaultAddress);
  return vault.deposit(assets, receiver);
}

// Withdraw assets from vault
import { VaultABI as IVault } from "./abis";
function main(vaultAddress: Address, assets: Uint256, receiver: Address, owner: Address): Uint256 {
  const vault = IVault.at(vaultAddress);
  return vault.withdraw(assets, receiver, owner);
}

// Redeem shares for assets
import { VaultABI as IVault } from "./abis";
function main(vaultAddress: Address, shares: Uint256, receiver: Address, owner: Address): Uint256 {
  const vault = IVault.at(vaultAddress);
  return vault.redeem(shares, receiver, owner);
}

// Preview deposit
import { VaultABI as IVault } from "./abis";
function main(vaultAddress: Address, assets: Uint256): Uint256 {
  const vault = IVault.at(vaultAddress);
  return vault.previewDeposit(assets);
}

// Convert assets to shares
import { VaultABI as IVault } from "./abis";
function main(vaultAddress: Address, assets: Uint256): Uint256 {
  const vault = IVault.at(vaultAddress);
  return vault.convertToShares(assets);
}
```

## ABI Methods
### VaultABI
- `deposit(uint256,address)` - Deposit assets. Params: assets (underlying token amount), receiver (share recipient). Returns shares minted
- `mint(uint256,address)` - Mint exact shares. Params: shares (exact shares to mint), receiver. Returns assets required
- `withdraw(uint256,address,address)` - Withdraw by asset amount. Params: assets (amount to withdraw), receiver, owner (share owner). Returns shares burned
- `redeem(uint256,address,address)` - Redeem shares. Params: shares (to burn), receiver, owner. Returns assets received
- `totalAssets()` - Total assets managed by vault. Returns uint256
- `convertToShares(uint256)` - Convert assets to shares (current rate). Params: assets. Returns shares
- `convertToAssets(uint256)` - Convert shares to assets (current rate). Params: shares. Returns assets
- `previewDeposit(uint256)` - Preview deposit (includes fees). Params: assets. Returns shares
- `previewMint(uint256)` - Preview mint cost. Params: shares. Returns assets needed
- `previewWithdraw(uint256)` - Preview withdrawal. Params: assets. Returns shares to burn
- `previewRedeem(uint256)` - Preview redemption. Params: shares. Returns assets received
- `maxDeposit(address)` - Max deposit allowed. Params: receiver. Returns max assets
- `maxMint(address)` - Max mint allowed. Params: receiver. Returns max shares
- `maxWithdraw(address)` - Max withdrawal allowed. Params: owner. Returns max assets
- `maxRedeem(address)` - Max redemption allowed. Params: owner. Returns max shares
- `asset()` - Underlying asset address. Returns address

## Notes
- Standard interface (EIP-4626) - protocols that implement it: sDAI, sfrxETH, pufETH, Yearn V3, sUSDe, Beefy vaults
- deposit/withdraw operate in asset terms; mint/redeem operate in share terms
- preview* functions account for fees; convert* are pure exchange rate calculations
- Approve the underlying asset to the vault before calling deposit
- owner parameter in withdraw/redeem allows withdrawal on behalf of another address (requires approval)
- Share price typically increases over time as yield accrues
